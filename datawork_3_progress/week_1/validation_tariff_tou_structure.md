# tariff_tou.csv 구조 검증 메모

- **작성일**: 2026-04-09
- **검증 범위**: 구조적 완전성만 (비용 계산식 타당성, 배출계수 파일은 이번 범위 외)

---

## 1. 파일 역할

시간대별 전기 요금표(TOU: Time-of-Use)로, ESS 충방전 비용 계산의 가장 앞단 입력 파일이다.
월(month)과 시간(hour)을 키로 삼아 단위 전력량 요금(`rate_won_per_kWh`)과 기본요금(`base_charge_won_per_kW`)을 조회하는 lookup table 역할을 한다.
이후 datetime 매핑, 시간대별 비용 계산, 더미 테스트 시나리오 생성의 기준이 된다.

---

## 2. 컬럼 구조

| 컬럼명 | 설명 | 예시 값 |
|---|---|---|
| `month` | 월 (1~12, 정수) | 1, 7, 12 |
| `hour` | 시간 (0~23, 정수) | 0, 9, 22 |
| `season` | 계절 구분 | 동계, 하계, 봄가을 |
| `load_type` | 부하 유형 | 경부하, 중간부하, 최대부하 |
| `rate_won_per_kWh` | 시간대별 전력량 요금 (원/kWh) | 52.5, 101.8, 162.2 |
| `base_charge_won_per_kW` | 기본요금 (원/kW) | 8220 |

총 6개 컬럼, 결측치 없음.

## 3. 월별 24시간 충족 여부

- **전체 행 수**: 288행 = 12개월 × 24시간 (기대값 일치 ✓)
- **월별 행 수**: 전 월(1~12월) 각 24행, 편차 없음
- **시간 범위**: 전 월에서 hour 0~23 완전 충족, 누락 없음
